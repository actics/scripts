ВИРТУАЛЬНАЯ МАШИНА (ПРИНЦИПЫ ФОН НЕЙМАНА).

   Данная виртуальная машина (VirMash.js) реализована в контексте архитектурных принципов фон Неймана.
   Так же в комплекте находятся файлы с кодами программ, написанных для данной виртуальной машины:
      1. Программа вычисляющая факториал (FCTRL.txt).
      2. Программа вычисляющая НОД 2х чисел (GCD.txt).

Запуск.

   Запуск осуществляется через консоль. Для запуска необходимо выполнить следующие действия:
      1. Ввести команду: cd /d "<Путь к папке с файлом VirMash.js>" без <>.
      2. Ввести команду: cscript VirMash.js <Имя файла с кодом исполняемой программы> без <>.
   Далее следует вводить данные по тому правилу, которое задает исполняемая программа.

Язык. 

   Вы можете самостоятельно написать программу для данной виртуальной машины. Вот список доступных команд:
      READ - команда, читающая число с клавиатуры, и помещающая его в ячейку памяти с номером, записанным в следующей после команды строчке.
      WRITE - команда, выводящая на экран данные из ячейки памяти с номером, который указан в следующей после команды строчке.
      END - команда, завершающая исполнение кода.
      ENLARGE - команда, увеличивающая на единицу число, записанное в ячейке памяти с номером, записанным в следующей после команды строке.
      MULTIPLICATION - команда, перемножающая два числа x и y, записанные в ячейках памяти с номерами указанными в первой и второй после
         команды строчке соответственно. В третей после команды строчке находится намер ячейки памяти, в которую необходимо положить
         результат операции.
      MODULO - команда, считающая остаток от деления числа x на число y, коротые расположены в ячейках памяти с номерами, указанными в
         первой и второй строчках после команды соответственно. В третей после команды строчке расположен номер ячейки памяти, в
         которую необходимо положить результат.
      GOTO - команда, осуществляющая безусловный переход на строчку с номером, расположенном в следующей после команды строчке.
      IF - команда, осуществляющая условный переход. В первой после команды строчке находится номер ячейки памяти, в которой лежит число,
         которое необходимо сравнить с числом, лежащим в ячейке памяти с номером находящимся в третей после команды строчке. Сравнение
         осуществляется по знаку, находящемуся во второй после команды строчке. Доступные знаки: > < >= <= = != . В четвертой после
         команды строке находится номер строки, на которую нужно перейти при выполнении условия.
   Код необходимо оформлять в файл с расширением .txt.

Возможные проблемы.

   В данной виртуальной машине не предусмотрена защита от дурака, поэтому написанный код для машины должен быть однозначно с верным
   синтаксисом, иначе машина сломается.

Дополнение 1. Исполнение программы вычисления факториала (FCTRL.txt).

   После запуска этого кода введите число. Скоро программа выведет в ответ число, которое является факториалом вашего.

Дополнение 2. Исполнение программы вычисления НОД 2х чисел (FCTRL.txt).

   После запуска этого кода введите число, а затем ещё одно. Скоро программа выведет в ответ число, которое является НОД ваших.
