КОДИРОВАНИЕ ХАФФМАНА.

   Данная программа представляет собой реализацию алгоритма сжатия текстовой информации, разработаного в 1952 году Дэвидом Хаффманом.
   Программа способна закодировать текст и наоборот.

Запуск.

   Программа запускается из консоли, по следующим правилам:
      1. Необходимо зайти в каталог с программой, для этого введите cd /d "<Путь к папке с файлом Huffman.js>"
      2. Для запуска введите cscript Huffman.js <действие> <вход> <выход> где: 
            <действие> принимает значения:
               -help - Вызов справки (для этого действия вводить <вход> и <выход> не нужно).
               -EnCode - Кодирование.
               -DeCode - Декодирование.
            <вход> - файл, из которого программе подается текст (должен обязательно быть на диске).
            <выход> - файл, в который выдается выход программы (при отсутствии создастся автоматически).

Возможные проблемы.

   На данный момент проблем не обнаружено.
